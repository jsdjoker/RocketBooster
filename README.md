# 🚀 RocketBooster

RocketBooster is a fun physics-based rocket game built in Unity where players must carefully control their rocket, avoid obstacles, and reach the destination. Master your thrusters and reflexes to conquer increasingly challenging levels!

## 🎮 Controls

| Key | Action |
|------|--------|
| Space Bar | Activate rocket boosters |
| A | Rotate Left |
| D | Rotate Right |

## ✨ Features

- 🚀 Physics-based rocket movement
- 🎯 Challenging obstacle courses
- ⚡ Simple and responsive controls
- 🎮 Easy to learn, difficult to master
- 🏁 Reach the finish platform to complete levels
- 💥 Avoid collisions and improve your piloting skills

## 🛠 Built With

- Unity
- C#
- Visual Studio
